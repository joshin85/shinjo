Control Hub

Control Hub is an android application that is designed to allow users to take advantage of their mobile devices and use them as a proxy mouse and keyboard. Easy to use and set-up this software requires only that you have a wireless network and only asks access to your wifi. Both the server and the client are written using Java and a variety of third party libraries.


The program allows for custom script execution(you have to write the scripts) which can be included using the command maker
